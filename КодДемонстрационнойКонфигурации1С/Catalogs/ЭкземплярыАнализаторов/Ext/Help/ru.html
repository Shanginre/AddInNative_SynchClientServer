Справочник «Экземпляры анализаторов»

 

Справочник предназначен для хранения списка экземпляров анализаторов.

 

Переход к списку экземпляров анализаторов производится из раздела «Настройки» / Панель навигации / Группа «Справочники» / «Экземпляры анализаторов».

 

Редактирование информации об экземплярах анализаторов производится в отдельном диалоговом окне, открываемом из формы списка справочника экземпляров анализаторов.

 

При этом указывается информация:

·        Код. Код экземпляра анализатора. Заполняется автоматически.

·        Наименование. В данном поле указывается наименование экземпляра анализатора. Требуется обязательное заполнение.

·        Тип анализатора. В данном поле указывается тип анализатора, который выбирается из списка типов, сформированного в разделе «Настройки» / Панель навигации / Группа «Настройки системы» / «Тип анализатора».

·        Режим сохранения результатов. Группа полей:

o       Сохранять результаты автоматически. Признак автоматического сохранения результатов для данного экземпляра анализатора. Для автоматического сохранения результатов установите флажок.

o       Обновлять заполненные результаты. Признак обновления заполненных результатов для данного экземпляра анализатора. Для обновления заполненных результатов установите флажок.

o       Сохранять только заказанные. Признак сохранения только заказанных результатов для данного экземпляра анализатора. Для сохранения только заказанных результатов установите флажок.

o       Окно поиска заявок (в днях). В данном поле указывается временной интервал

(в днях), в котором ищется заявка для автоматического сохранения пришедших результатов.

o       Сопоставление результатов. В данном поле указывается, по какому принципу необходимо сопоставлять результаты. Выбирается одно из двух значений: «По номеру» или «По штрихкоду».

·        Методики. Данная табличная часть может быть заполнена автоматически методиками, зарегистрированными для данного типа анализатора (кнопка «Заполнить методики» командной панели табличной части) либо методом ручного ввода. Поля табличной части:

o       N. Номер записи методики. Заполняется автоматически.

o       Методика. В данном поле укажите наименование методики для экземпляра анализатора с данным типом анализатора. Выбирается из списка методик, для которых в разделе «Настройки» / Панель навигации / Группа «Настройки системы» / «Методики» указан данный тип анализатора.
